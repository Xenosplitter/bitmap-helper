Bitmap Helper is a *very* simple web-based drawing tool for pixel bitmaps.

The intention of the tool is to provide color shortcodes that can be saved to a microcontroller to display on a NeoPixel matrix. Specifically, a 8x32 matrix of [WS2812B](https://www.ledyilighting.com/wp-content/uploads/2025/02/WS2812B-datasheet.pdf) pixels. I bought my strip [here](https://www.amazon.com/WESIRI-WS2812B-Flexible-Individually-Addressable/dp/B07P5TSKX8) (non-affiliated).

I've found WS2812B pinks look very washed out, try using magenta instead.
